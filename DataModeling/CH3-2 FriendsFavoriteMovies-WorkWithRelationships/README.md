# Friend's Favorite Movies: Work with relationships

Extend the Friends’ Favorite Movies project from [Create, update, and delete data](https://developer.apple.com/tutorials/develop-in-swift/create-update-and-delete-data) to relate friends to movies and to let people search for movies by title.

Most apps have multiple types of model data that are related in some way, such as calendars and appointments. To connect friends and movies in your app, you’ll establish a relationship between the two model types and update the app to display and edit related data. Then you’ll implement search by building a custom query.

## Overview

These resources are associated with the [Work with relationships](https://developer.apple.com/tutorials/develop-in-swift/work-with-relationships) tutorial. This is part of the [Data Modeling](https://developer.apple.com/tutorials/develop-in-swift/welcome-to-data-modeling) tutorials from [Develop in Swift](https://developer.apple.com/tutorials/develop-in-swift) by Apple.
